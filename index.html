xrdocs-images

Repository to host images for xrdocs

View on GitHub Download .zip Download .tar.gz

To upload your images to github for hosting: 

      
    
       
  • Browse to https://github.com/xrdocs/xrdocs-images/upload/gh-pages/assets/images
    • 
       
  • If you need to be more structured, then upload images based on your doc-type: 
 Blogs:
     https://github.com/xrdocs/xrdocs-images/upload/gh-pages/assets/blog-images
 Tutorials:
     https://github.com/xrdocs/xrdocs-images/upload/gh-pages/assets/tutorial-images
 Techdocs:
     https://github.com/xrdocs/xrdocs-images/upload/gh-pages/assets/techdoc-images
    • 

       
  •  Drag and drop your images from your workstation and commit 
    • 
       
  •  The github link: 
     https://github.com/xrdocs/xrdocs-images/tree/gh-pages/assets/
 directly corresponds to the web server at:
     https://xrdocs.github.io/xrdocs-images/assets/
    • 
       
  •  Your images will now be available at 
 https://xrdocs.github.io/xrdocs-images/assets/images/<image-name.jpg/png/jpeg>
    •  
      
  •  Now, to insert the image in any markdown file, use: 
 ![<image context>](https://xrdocs.github.io/xrdocs-images/assets/images/<image-name.jpg/png/jpeg>)
    • 
     
  • If you uploaded to a specific directory like https://github.com/xrdocs/xrdocs-images/upload/gh-pages/assets/tutorial-images
 Then the image may be refrenced as:
 ![<image context>](https://xrdocs.github.io/xrdocs-images/assets/images/<image-name.jpg/png/jpeg>)
 
    •
